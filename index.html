Smart Traffic-Aware
Shortest Path Algorithm
for Dynamic Road Networks

A modified Dijkstra implementation incorporating real time traffic conditions alongside static road distances to compute more realistic navigation routes.

Debaditya Malakar · RA2411028010034  |  Monish Ravibalaji · RA2411028010013  |  Syed Gaffar Mohammed · RA2411028010047  | 
Course project Algorithm Design & Analysis
Traditional shortest path algorithms optimise routes on distance alone. Real world navigation must account for traffic congestion, incidents, and variable road conditions. This implementation modifies edge weights as w = distance + traffic factor, enabling the algorithm to prefer less congested routes over geometrically shorter ones.
Effective Edge Weight
w(u,v) = dist(u,v) + traffic(u,v)
